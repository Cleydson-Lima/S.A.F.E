Sistema de avaliação física escolar

Site desenvolvido para o uso exclusivo do professor de educação física.
Ferramenta que irá auxilia-lo nos testes feitos com alunos.

Fazer Cadastro

Serviços Oferecidos

O objetivo é oferecer, através desse site, um auxílio para que professor de educação física possa armazenar dados de testes de aptidões físicas de alunos, realizar cálculos, comparações e muito mais.

Dados dos Testes

Após a bateria de teste de aptidão física realizada pelo professor, conforme o manual Proesp, o sistema irá armazenar todas as informações obtidas sobre o aluno.

Cálculos

Através dos dados obtidos o sistema, automaticamente, vai realizar cáculos dos dados dos alunos, como por exemplo Índice de Massa Corporal (IMC) e o Índice de Relação Cintura Quadril (IRCQ).

Escala Categórica

O sistema, de acordo com os cálculos realizados, irá exibir uma escala categórica de dois degraus: "Zona de Risco a Saúde" e "Zona Saudável", na qual permite o professor avaliar os seus alunos.

Comparações

Após cálculos, escalas e gráficos, o sistema oferece a opção de comparar os dados dos alunos entre os primeiros e últimos testes realizados, para acompanhar a sua evolução. Além desta comparção indivdual, temos também a comparação em uma escala ampla, permitindo visualizar a diferença entre salas e escolas.
